== Description ==

Enables GeoRiot Auto Link conversion of iTunes and Amazon URL's for all pages.

For more information visit:
http://support.georiot.com/support/solutions/articles/72229-link-building

== Installation ==

1. Upload `georiot-autolinker` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your GeoRiot TSID at (Admin > Settings > GeoRiot Autolinker)
